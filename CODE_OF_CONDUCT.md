# ProofMoney Code of Conduct

ProofMoney community spaces should focus on protocol integrity.

Expected discussion areas:

- documentation;
- verification;
- protocol design;
- open-source review;
- testing;
- security;
- user education;
- risk awareness;
- infrastructure.

Forbidden community behavior includes:

- harassment;
- scams;
- phishing;
- spam;
- impersonation;
- trading signal groups;
- paid shilling;
- private sale solicitation;
- sharing malicious links;
- encouraging users to disclose secrets;
- spreading fake announcements;
- manipulating expectations around PRM value.

ProofMoney is not a trading community.

It is an open protocol initiative for verifiable money integrity.
